## 🎨 自定义说明

### 更换熊猫图片
在 `index.html` 中找到对应的 `<img>` 标签，修改 `src` 属性即可。

### 修改配色
在 `styles.css` 的 `:root` 部分修改 CSS 变量：
```css
:root {
    --primary-color: #2c3e50;
    --bamboo-green: #2ecc71;
    --accent-color: #f39c12;
}
```

### 添加熊猫成员
复制 `.panda-card` 的 HTML 结构，修改内容即可。
